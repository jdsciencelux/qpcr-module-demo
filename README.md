# Interactive qPCR Training Module  
### Built with Articulate Storyline 360

An interactive eLearning module designed to teach foundational and applied concepts of quantitative Polymerase Chain Reaction (qPCR) through scenario-based learning, visual walkthroughs, and learner-driven interaction.

This project demonstrates the ability to translate technical biomedical content into accessible, engaging digital instruction for laboratory, healthcare, and STEM education environments.

---

## Project Overview

This module was developed as part of a professional instructional design portfolio focused on technical and scientific training solutions.

The learning experience introduces users to:
- Core qPCR concepts
- Laboratory workflow procedures
- Amplification curve interpretation
- Threshold cycle (Ct) value analysis
- Common laboratory errors and troubleshooting
- Experimental accuracy and data interpretation

The module was designed with an emphasis on:
- learner engagement
- accessibility
- cognitive scaffolding
- interactive reinforcement
- real-world application

---

## Learning Objectives

By the end of the module, learners will be able to:

- Explain the purpose and function of qPCR
- Identify key stages of the qPCR workflow
- Interpret amplification plots and Ct values
- Distinguish between valid and invalid experimental outcomes
- Apply troubleshooting strategies to common qPCR issues

---

## Instructional Design Strategy

This project uses several instructional design methodologies commonly applied in technical workforce training and STEM education:

- Scenario-based learning
- Progressive disclosure
- Interactive knowledge checks
- Visual process mapping
- Guided learner feedback
- Applied problem-solving exercises

The experience was intentionally designed to reduce cognitive overload while maintaining scientific rigor for technical learners.

## Design Decisions

### Why Scenario-Based Learning?
qPCR concepts can become highly abstract when presented passively. Scenario-based interaction was used to contextualize laboratory decisions and reinforce practical application.

### Why Interactive Visualization?
Many learners struggle with interpreting amplification data and Ct values. Interactive visual elements were incorporated to improve conceptual retention and reduce intimidation around technical material.

### Why This Project Matters
Scientific and biomedical training often suffers from dense, text-heavy delivery. This module demonstrates how instructional design can improve accessibility and learner engagement within highly technical subject matter.
